FLEX BOX

Es una tecnología de diseño en CSS que permite crear diseños flexibles y dinámicos en la página web. Con Flexbox, es posible controlar el tamaño, la posición y el orden de los elementos en una página web de manera más eficiente que con los métodos de diseño tradicionales.

Flexbox es una forma de diseño de cajas, donde cada elemento de la página web se trata como una caja. Estas cajas se organizan en una línea principal (main axis) y una línea transversal (cross axis) dependiendo de la dirección en la que se configure el contenedor.

Las propiedades principales de Flexbox son flex-container y flex-item. Para aplicar Flexbox en un contenedor, se debe definir su propiedad como un contenedor flex utilizando display: flex. Esto indica que todos los elementos dentro del contenedor serán elementos flex. Luego, se pueden utilizar diferentes propiedades, como justify-content, align-items, flex-wrap y flex-direction, para ajustar la posición y el tamaño de los elementos dentro del contenedor.
